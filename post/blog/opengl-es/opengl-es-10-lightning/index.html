一、冯氏光照模型（Phone Lightning Model）

对于我们世界而言，光照是许多因素组成的，但是在变成的世界中，我们以冯氏光照模型便可以将光照表达出来：

其中冯氏光照模型由 3 个分量组成：

  • 环境光照(Ambient Lighting)：即使在黑暗的情况下，世界上也仍然有一些光亮(月亮、一个来自远处的光)，所以物体永远不会是完全黑暗的。我们使用环境光照来模拟这种情况，也就是无论如何永远都给物体一些颜色。
  • 漫反射光照(Diffuse Lighting)：模拟一个发光物对物体的方向性影响(Directional Impact)。它是冯氏光照模型最显著的组成部分。面向光源的一面比其他面会更亮。
  • 镜面光照(Specular Lighting)：模拟有光泽物体上面出现的亮点。镜面光照的颜色，相比于物体的颜色更倾向于光的颜色。

通过设置物体的光照相关参数，最终将各个分量之和与物体颜色相乘，即可得到最后的颜色：

    color = (ambient + diffuse + specular) * originColor;

假设现在的物体颜色（objColor）是绿色，灯光颜色（lightColor）是白色，不加入任何光照是这种效果：

下面开始介绍冯氏光照模型的 3 种光照分量对物体的影响。

环境光照(Ambient Lighting)

环境光分量 (ambient) = 环境光强度 (ambientStrength) * 灯光颜色 (lightColor)

反映的是遍布在物体周围的光，作用于所有的面

PS：环境光强度 (ambientStrength) 通常是一个很小的值，比如 0.1，所以仅设置该分量会导致物体十分暗

仅加了环境光分量的 shader：

fragment shader：


        // Ambient
        highp vec3 ambient = ambientStrength * lightColor;

        color = (ambient) * originColor;
        gl_FragColor = vec4(color, 1.0);

设环境光强度 (ambientStrength) 为 0.1，则效果如下

仔细看，还是可以看出来，物体的原本颜色是绿色，只是非常暗。

漫反射光照(Diffuse Lighting)

漫反射是指 光源照射到物体片段上反射的颜色，当光源与物体片段的法线夹角越小，则反射的光越多即越亮

法线是指垂直于平面的单位向量

由上面的定义可以看出，漫反射是作用于所有片段的。

因为计算漫反射还需要 法向量 & 当前片段的位置，因此要修改顶点着色器去计算出这两个值

vertex shader

attribute vec3 position;
attribute vec3 normal;

uniform mat4 modelMatrix;
uniform mat4 viewMatrix;
uniform mat4 projectionMatrix;

varying vec3 outNormal; // 输出的法向量
varying vec3 outFragmentPosition; // 输出的片段位置

void main() {
    
    gl_Position = projectionMatrix * viewMatrix * modelMatrix * vec4(position, 1.0);
    outNormal = vec3(modelMatrix * vec4(normal, 0.0));
    outFragmentPosition = vec3(modelMatrix * vec4(position, 1.0));
}

其中需要：

  • 定义好物体的法向量 normal 传进来顶点着色器

  • 将变换矩阵乘 normal 得出变换后的法向量 outNormal

  • 将变换矩阵乘 position 得出变换后的片段位置 outFragmentPosition

接着计算漫反射需要使用到 入射光线的方向 lightDir，以及归一化法线，两者点乘（dot）即可以得到他们的夹角，也就是漫反射程度。

fragment shader

        // Diffuse
        highp vec3 norm = normalize(outNormal);
        highp vec3 lightDir = normalize(lightPos - outFragmentPosition);
        
        highp float diff = max(dot(norm, lightDir), 0.0);
        highp vec3 diffuse = diff * lightColor;

        color = (diffuse) * originColor;
        gl_FragColor = vec4(color, 1.0);

镜面光照(Specular Lighting)

镜面光照和人的观察方向有关，即物体的颜色会随着我们看的颜色而不同，像镜子反射光线一样。

类似 diffuse，我们有视线的方向 （eyeDir），即观察方向和片段位置之差的归一化向量；反射反向（reflectDir），是由光线方向和法线方向决定，具体可以看代码：

fragment shader


        // Specular
        highp vec3 eyeDir = normalize(eyePos - outFragmentPosition); // 视线方向
        highp vec3 reflectDir = reflect(-lightDir, norm); // 光线反射的方向
        
        highp float spec = pow(max(dot(eyeDir, reflectDir), 0.0), 32.0);
        highp vec3 specular = specularStrength * spec * lightColor;

        color = (specular) * originColor;
        gl_FragColor = vec4(color, 1.0);

此处需要注意：

  • 32.0 代表了物体的发光值（shiness），该值代表了物体的反光能力，该值越大，光聚点越小（即散点越少）。

  • specularStrength 是镜面光强度，为了防止光线反射过度夸张，此处设为 0.5

最终实现的效果如下：光线聚集到黑色物体的一个点上反射，由于没有环境光和漫反射，所以看不到物体，只有一个聚光点

有兴趣的可以调整一下 shiness 看看效果。

完整的 shader 代码：

vertex shader (Light.vsh)：

attribute vec3 position;
attribute vec3 normal;

uniform mat4 modelMatrix;
uniform mat4 viewMatrix;
uniform mat4 projectionMatrix;

varying vec3 outNormal; // 输出的法向量
varying vec3 outFragmentPosition; // 输出的片段位置

void main() {
    
    gl_Position = projectionMatrix * viewMatrix * modelMatrix * vec4(position, 1.0);
    outNormal = vec3(modelMatrix * vec4(normal, 0.0));
    outFragmentPosition = vec3(modelMatrix * vec4(position, 1.0));
}

fragment shader（Light.fsh）：

varying highp vec3 outNormal; // 输出的法向量
varying highp vec3 outFragmentPosition; // 输出的片段位置

uniform highp vec3 originColor; // 物体的颜色

uniform highp int needLight; // 是否需要光照（光源不需要） If rendering "Lamb", this value is 0; else 1;

uniform highp vec3 lightColor; // 灯的颜色
uniform highp float ambientStrength; // 环境光强度

uniform highp vec3 lightPos; // 灯的位置

uniform highp vec3 eyePos; // 摄像机（眼睛）的位置

uniform highp float specularStrength; // 镜面光强度

void main() {
    
    highp vec3 color;
    
    if (needLight != 0) { // The Object, calculate the final color affected by light
        
        // Ambient
        highp vec3 ambient = ambientStrength * lightColor;
        
        // Diffuse
        highp vec3 norm = normalize(outNormal);
        highp vec3 lightDir = normalize(lightPos - outFragmentPosition);
        
        highp float diff = max(dot(norm, lightDir), 0.0);
        highp vec3 diffuse = diff * lightColor;
        
        // Specular
        highp vec3 eyeDir = normalize(eyePos - outFragmentPosition); // 视线方向
        highp vec3 reflectDir = reflect(-lightDir, norm); // 光线反射的方向
        
        highp float spec = pow(max(dot(eyeDir, reflectDir), 0.0), 32.0);
        highp vec3 specular = specularStrength * spec * lightColor;
        
        
        // Combine all kind of colors
        color = (ambient + diffuse + specular) * originColor;
        
    } else { // The "Lamb",  use originColor
        
        color = originColor;
        
    }
    
    gl_FragColor = vec4(color, 1.0);
    
}

二、完整流程

该 demo 选择绘制两个立方体，一个作为光源一个作为物体，大致的流程和 画一个立方体 一致，但是有如下需要注意的点：

1、 绘制两个立方体

  • 需要有两个 VAO，分别作为 light 和 obj 的 VAO

    GLuint _lightVAO, _objVAO;

- (void)setupVAOs {
    _lightVAO = [self createVAO];
    _objVAO = [self createVAO];
}

  • 需要两个 ModelMatrix，分别定义了 light 和 obj 的本地坐标变换 （ViewMatrix 和 ProjectionMatrix 使用同一个）

    其中 light 的 ModelMatrix 是一直不变的；obj 的 ModelMatrix 是会不断更新旋转角度的

    
GLKMatrix4 _lightModelMatrix, _objModelMatrix, _viewMatrix, _projectionMatrix;

#pragma mark - Set up light Cube (By Model matrix)

- (void)setupLightCube {
      
    GLKMatrix4 lightTranslation = GLKMatrix4MakeTranslation(_lightPosition.x,
                                                            _lightPosition.y,
                                                            _lightPosition.z);
    GLKMatrix4 lightScale = GLKMatrix4MakeScale(_lightScale, _lightScale, _lightScale);
      
    _lightModelMatrix = GLKMatrix4Multiply(lightTranslation, lightScale);
      
}

#pragma mark - Rotate the Cube (By Model matrix)

- (void)rotateCube {
      
    GLKMatrix4 objTranslation = GLKMatrix4MakeTranslation(_objPosition.x, _objPosition.y, _objPosition.z);
    GLKMatrix4 objRotation = GLKMatrix4MakeRotation(GLKMathDegreesToRadians(_objRotateAngle), 1, 1, 1);
    GLKMatrix4 objScale = GLKMatrix4MakeScale(_objScale, _objScale, _objScale);
      
    // 从后面读起：先 scale -> 再 rotate -> 最后 translate
    _objModelMatrix = GLKMatrix4Multiply(GLKMatrix4Multiply(objTranslation, objRotation), objScale);
      
    _objRotateAngle += 1;
      
    if (_objRotateAngle > 360) {
        _objRotateAngle = 0;
    }
      
}

  • 渲染两个 Cube

    需要使用两个 program，分别对应 light 和 obj 的渲染程序，不过使用的是同一套的 shader，在具体 render 的时候，传入 needLight 来判断是否使用光照模型即可

        // Programs
    GLuint _lightProgram, _objProgram;

    渲染两个 cube

    - (void)render: (CGSize)clearSize {
  
    // 0. Bind the FBO & Clear the FBO
      
    glBindFramebuffer(GL_FRAMEBUFFER, _FBO);
    [self clearFBO: clearSize];
      
    // 1. Render light
      
    glUseProgram(_lightProgram);
    glUniformMatrix4fv(_modelMatrixUniformLoc, 1, GL_FALSE, _lightModelMatrix.m);
    glUniformMatrix4fv(_viewMatrixUniformLoc, 1, GL_FALSE, _viewMatrix.m);
    glUniformMatrix4fv(_projectionMatrixUniformLoc, 1, GL_FALSE, _projectionMatrix.m);
      
    glUniform3f(_originColorUniformLoc, _lightColor.r, _lightColor.g, _lightColor.b); // 物体颜色
    glUniform1i(_needLightUniformLoc, 0); // 是否需要光照
    glUniform3f(_lightColorUniformLoc, _lightColor.r, _lightColor.g, _lightColor.b); // 光照颜色
    glUniform1f(_ambientStrengthUniformLoc, _lightAmbientStrength); // 环境光强度
    glUniform1f(_specularStrengthUniformLoc, _specularStrength); // 镜面光强度
    glUniform3f(_lightPosUniformLoc, _lightPosition.x, _lightPosition.y, _lightPosition.z); // 光源的位置
    glUniform3f(_eyePosUniformLoc, _eyePostion.x, _eyePostion.y, _eyePostion.z); // 眼睛（摄像机）的位置
      
    glBindVertexArray(_lightVAO); // 使用 VAO 的好处，就是一句 bind 来使用对应 VAO 即可
    glDrawArrays(GL_TRIANGLES, 0, sizeof(vertices) / 3);
      
    // 2. Render Object
      
    glUseProgram(_objProgram);
    glUniformMatrix4fv(_modelMatrixUniformLoc, 1, GL_FALSE, _objModelMatrix.m);
    glUniformMatrix4fv(_viewMatrixUniformLoc, 1, GL_FALSE, _viewMatrix.m);
    glUniformMatrix4fv(_projectionMatrixUniformLoc, 1, GL_FALSE, _projectionMatrix.m);
      
    glUniform3f(_originColorUniformLoc, _objColor.r, _objColor.g, _objColor.b); // 物体颜色
    glUniform1i(_needLightUniformLoc, 1); // 是否需要光照
    glUniform3f(_lightColorUniformLoc, _lightColor.r, _lightColor.g, _lightColor.b); // 光照颜色
    glUniform1f(_ambientStrengthUniformLoc, _lightAmbientStrength); // 环境光强度
    glUniform1f(_specularStrengthUniformLoc, _specularStrength); // 镜面光强度
    glUniform3f(_lightPosUniformLoc, _lightPosition.x, _lightPosition.y, _lightPosition.z); // 光源的位置
    glUniform3f(_eyePosUniformLoc, _eyePostion.x, _eyePostion.y, _eyePostion.z); // 眼睛（摄像机）的位置
      
    glBindVertexArray(_objVAO); // 使用 VAO 的好处，就是一句 bind 来使用对应 VAO 即可
    glDrawArrays(GL_TRIANGLES, 0, sizeof(vertices) / 3);
}

    • a、先 bind FBO & Clear FBO

    • b、渲染 light：

      • 选择 light 的 program
      • 传入 light shader 的 uniforms （注意 needLight 传入 0）
      • 选择 light 的 VAO
      • glDrawArrays 画出 light 的 cube 到 FBO

    • c、渲染 cube：

      • 选择 cube 的 program
      • 传入 cube shader 的 uniforms （注意 needLight 传入 1）
      • 选择 light 的 VAO
      • glDrawArrays 画出 cube 的 cube 到 FBO
