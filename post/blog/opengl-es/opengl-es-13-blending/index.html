OpenGL ES in iOS - 13 混合 blending

 发表于 May 29, 2020  |  2 分钟  |  272 个字  |  Kim
  • 一、介绍
  • 二、OpenGL ES 中的混合开关
  • 三、混合函数
  • 四、混合因子（factor）

当两个物体的透明度 alpha 不为 1 的时候，在同一个位置的显示颜色通常要按照一定的规则在两个物体的颜色中取值，最后将得到的颜色显示，这个过程被称 混合 blending，使用的是 物体透明技术。

本文使用两个半透明物体进行混合，代码可以查看 demo 仓库 的 11.混合blend。

一、介绍

混合发生在 片元着色器 fragment shader 之后的 逐片元操作 Per-Fragment Operations 过程中。

在对物体片元都上色之后，会将通过一系列测试后的片元与 原来绘制的颜色 进行混合，然后在通过抖动之后会把最后的颜色输出到 FrameBuffer 中。

二、OpenGL ES 中的混合开关

混合的开关是上下文保存的状态，可通过下述方式设置开关混合

  • 开启混合：glEnable(GL_BLEND);

  • 关闭混合：glDisable(GL_BLEND);

三、混合函数

在 OpenGL ES 中给予我们自由去设置混合规则，使用如下函数可以设置混合函数：

glBlendFunc (GLenum sfactor, GLenum dfactor)

参数：

  • sfactor Source Factor 源因子。设置了对源颜色的 alpha 值影响。

  • dfactor Destination Factor 目标因子。设置了对目标颜色的 alpha 影响。

因为混合是对 两个颜色 的操作，所以输入还应该包括有两个 颜色向量：

  • sColor Source Color 源颜色向量。这是来自纹理的本来的颜色向量。（即纹理颜色 Fragment Shader 的 gl_FragColor）

  • dColor Destination Color 目标颜色向量。这是储存在颜色缓冲中当前位置的颜色向量。（即 FrameBuffer 中原有的颜色）

混合公式如下定义：

resColor = sColor * sFactor + dColor * dFactor

注意，结果会 clamp 到 0 ~ 1，大于 1 取 1，小于 0 取 0。

四、混合因子（factor）

混合因子可以看作是一个 四维向量，与颜色的相乘过程可以看作是两个向量的积。

混合因子使用 GLenum 定义，设：

  • 源颜色向量 SourceColor sC = (sR, sG, sB, sA)

  • 目标颜色向量 DestinationColor sD = (dR, dG, dB, dA)

混合因子对应向量表达式有如下关系：

混合因子向量表达式向量展开式
GL_ZERO0(0, 0, 0, 0)
GL_ONE1(1, 1, 1, 1)
GL_SRC_COLORsC(sR, sG, sB, sA)
GL_DST_COLORsD(dR, dG, dB, dA)
GL_ONE_MINUS_SRC_COLOR1 - sC(1-sR, 1-sG, 1-sB, 1-sA)
GL_ONE_MINUS_DST_COLOR1 - sD(1-dR, 1-dG, 1-dB, 1-dA)
GL_SRC_ALPHAsA(sA, sA, sA, sA)
GL_DST_ALPHAdA(dA, dA, dA, dA)
GL_ONE_MINUS_SRC_ALPHA1 - sA(1-sA, 1-sA, 1-sA, 1-sA)
GL_ONE_MINUS_DST_ALPHA1 - sD(1-dA, 1-dA, 1-dA, 1-dA)
GL_SRC_ALPHA_SATURATEsat = min(sA, 1-dA)(sat, sat, sat, sat)

比如：

  • 先绘制一个绿色的不透明矩形 dColor = (0, 1, 0, 1)；

  • 再绘制一个红色的半透明颜色 sColor = (1, 0, 0, 0.5)；

  • 源因子 sFactor 为 GL_SRC_ALPHA sFactor = 0.5

  • 目标因子 dFactor 为 GL_ONE_MINUS_SRC_ALPHA dFactor = 1 - 0.5 = 0.5

最终颜色：

resColor = sColor * sFactor + dColor * dFactor
resColor = (1, 0, 0, 0.5) * 0.5 + (0, 1, 0, 1) * 0.5
resColor = (0.5, 0.5, 0, 0.75)

最终显示的颜色是一半绿色一半红色组合起来的 “深绿色”。
