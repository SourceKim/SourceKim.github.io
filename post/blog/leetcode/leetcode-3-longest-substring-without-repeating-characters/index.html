题目：

 * Given a string, find the length of the longest substring without repeating
 * characters.
 * 
 * 
 * Example 1:
 * 
 * 
 * Input: "abcabcbb"
 * Output: 3 
 * Explanation: The answer is "abc", with the length of 3. 
 * 
 * 
 * 
 * Example 2:
 * 
 * 
 * Input: "bbbbb"
 * Output: 1
 * Explanation: The answer is "b", with the length of 1.
 * 
 * 
 * 
 * Example 3:
 * 
 * 
 * Input: "pwwkew"
 * Output: 3
 * Explanation: The answer is "wke", with the length of 3. 
 * ⁠            Note that the answer must be a substring, "pwke" is a
 * subsequence and not a substring.

给出一个字符串，找出 最长的 & 没有重复字符的 子字符串

思路

双指针法

  • 为了满足 最长的： 通过左右指针的移动，找到 maxLength，maxLength = right - left

    此处可以使用 trick：让 left 指向目标的前一个，就不用使长度 +1，否则 length 应该为 right - left + 1

  • 为了满足 没有重复字符的：需要使用一个 map 记录下来字符最后出现的位置，key 是 字符，value 是最后出现的位置

    此处使用一个 trick：用一个 int 数组即可保存，此时可以将数组的 index 当做 key index = char - 'a'

    比如：“caba”，当遍历到 b 的时候，可以这样记录 vec[b - 'a'] = 2

如何移动指针？

从前往后遍历字符串，即移动 right 指针

如果当前遍历的字符 str[right] 出现过，则让 left 移动到 str[right] 最后出现的位置，即 vec[str[right]]，此时的长度应该为 right - left；如果没出现过，则让 left 指针保持原位

vec

如何找到最大？

在遍历的时候（移动 right 指针），且更新了 left 之后，要把当前的长度 right - left 和之前的最大值 maxLength 作对比，然后 maxLength 取较大者

注意

vec 初始化要为 -1，因为 vec[str[right]] 最小是 0

代码

class Solution {
public:
    int lengthOfLongestSubstring(string s) {

        /* 
        pos 的作用是：记录字符最后出现的位置
        key - 字符 
        value - 最后出现的位置 (-1 为从未出现)
         */
        vector<int> pos(256, -1);

        int left = -1; // 左边指针

        int maxLength = 0; // 最长长度

        for (int right = 0; right < s.length(); right++) { // 遍历每个字符

            /* 
                当前字符出现过 - 更新 left 到当前字符
                当前字符没出现过 - 不更新 left 的位置 （left 最小为 0， pos 初始化全是 -1
             */
            left = max(left, pos[s[right]]); 

            /* 
                记录当前字符出现的 pos
             */
            pos[s[right]] = right;

            /* 
                更新最大长度
             */
            maxLength = max(maxLength, right - left);

        }

        return maxLength;
    }
};
