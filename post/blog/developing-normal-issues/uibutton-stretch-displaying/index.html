UIButton 图片拉伸显示

 发表于 June 9, 2020  |  2 分钟  |  246 个字  |  Kim
  • 一、UIButton 的 UIImageView 尺寸的决定策略
  • 二、UIButton 拉伸显示
    • 1、修改 UIButton 内容的对齐方式

日常开发中，有时候的需求是 让按钮的图片拉伸填满按钮，有两种实现方式：

  • 1、修改 UIButton 内容的拉伸方式

  • 2、设置 ResizableImage

测试代码可以参考我的 仓库 的 UIButtonTileImage 工程。

一、UIButton 的 UIImageView 尺寸的决定策略

当不设置其他属性的时候，UIButton 中 UIImageView 的尺寸是根据 UIButton 的 size 和 我们图片的 size 决定的。

策略如下

  • 当 Button.size > Image.size，则 ImageView 的大小和 Image 的 size 一致。

    设有 Button.size == (300, 300)，Image.size == (200, 200)：

    此时 ImageView.size == Image.size == (200, 200)

  • 当 Button.size < Image.size 则 ImageView 的大小和 Button 的 size 一致。

    设有 Button.size == (50, 50)，Image.size == (200, 200)：

    此时 ImageView.size == Button.size == (50, 50)

二、UIButton 拉伸显示

当 Button.size > Image.size 的时候，ImageView 的 size 和 Image 的 size 会保持一致，此时如果需要将 ImageView 拉伸到和 Button 的 size 一致的时候，即如下效果：

有如下两个方式：

1、修改 UIButton 内容的对齐方式

UIControl 有如下两个属性，可以设置内容的对齐方式

  • contentVerticalAlignment 垂直方向的对齐方式

  • contentHorizontalAlignment 水平方向的对齐方向

有如下的定义：

    public enum ContentHorizontalAlignment : Int {

        
        case center = 0

        case left = 1

        case right = 2

        case fill = 3

        @available(iOS 11.0, *)
        case leading = 4

        @available(iOS 11.0, *)
        case trailing = 5
    }

设置为 fill 即可使 ImageView 充满 Button。

代码如下：

    private func tileButtonImage_0() {
        btn.contentHorizontalAlignment = .fill
        btn.contentVerticalAlignment = .fill
    }

2、设置 ResizableImage

该方式是将 Image 转化成 ResizableImage 实现的，可以调用如下 api：

    @available(iOS 6.0, *)
    open func resizableImage(withCapInsets capInsets: UIEdgeInsets, resizingMode: UIImage.ResizingMode) -> UIImage

参数：

  • capInsets Image 4 个方向填充的像素值

  • resizingMode 填充模式，有两个枚举

        public enum ResizingMode : Int {

      
    case tile = 0

    case stretch = 1
}

    • tile 平铺，如果图片太小，会进行 重复贴图，直至填满

    • stretch 拉伸，如果图片太小，会将图片 拉伸 直至填满

因此我们让其 inset 为 0，模式为 stretch 即可，具体代码如下：

    private func tileButtonImage_1() {
        
        var image = btn.image(for: .normal)
        image = image?.resizableImage(withCapInsets: .zero, resizingMode: .stretch)
        
        btn.setImage(nil, for: .normal)
        btn.setBackgroundImage(image, for: .normal)
    }

注意要设置为 UIButton 的 backgroundImage，否则无效。
