算法 - 归并排序

 Posted on July 20, 2018  |  3 minutes  |  443 words  |  Kim

归并排序的时间复杂度是 O(nlogn)，最坏情况也是 O(nlogn)

归并排序是 稳定 排序， 因为大小相同不做交换

PS: 稳定排序的定义是若 arr[i] == arr[j]，排序之后第 i & 第 j 个元素的 前后顺序不变。

归并排序很好地体现了 分治思想：

1. 分

将要排序的内容从中间 分开

2. 治

再将分开的数组进行两两合并，治之

3. 具体步骤如下：

若有数组 [9, 3, 5, 1, 6, 2, 8, 7]

先将其 分：

变成 2 组：

[9, 3, 5, 1] & [6, 2, 8, 7]

再分为 4 组：

[9 ,3] [5, 1] & [6, 2] [8, 7]

直到不可分割了，再进行对比 治

先对比 4 组

[3, 9] [1, 5] & [2, 6] [7, 8]

再两两对比成 2 组：

[1, 3, 5, 9] & [2, 6, 7, 8]

再合并成最终的 1 组

[1, 2, 3, 5, 6, 7, 8, 9]

因此要有两个函数，分别是 对应 分 & 治 两个功能：

分

使用递归的方式

  • 递归出口：left <= right

split(arr, left, right) {

    if left <= right {
        return
    }

    int mid = (left + right) / 2 // 先找到中点

    // 分成两组进行递归：
    split(arr, left, mid)
    split(arr, mid + 1, right)

    // 最后调用合并
    merge(arr, left, mid, right)
}

治，即合并两个数组

    merge(arr, left, mid, right) {

        // 先申请一个区域去存排序后的数，k 指向这块区域的第一个
        int tmp[] = new tmp[arr.length]
        int k = 0

        // 1. 先从两条链的头开始比对
        int i = left // 指向链 1 的第一个
        int j = mid + 1 // 指向链 2 的第一个

        while (i <= mid && j <= right) {
            // 将两条链的数据有序放入
            if (arr[i] < arr[j]) {
                tmp[k] = arr[i]
                k++
                i++
            } else {
                tmp[k] = arr[j]
                k++
                j++
            }
        }

        // 2. 此时 i 或者 j 有一个应该指到它对应的链的头了，
        // 这时候要把另外一条链的元素直接放入 tmp
        while (i <= mid) {
                tmp[k] = arr[i]
                k++
                i++
        }

        while (j <= right) {
                tmp[k] = arr[j]
                k++
                j++          
        }

        // 3. 最后将 tmp 逐个放回到 arr
        k = 0
        while (left <= right) {
            arr[left] = tmp[k]
            left++
            k++
        }

    }

完整代码 （C++）

class Solution2 {
public:
    int sortArray(int arr[]) {
        mergeSort(arr, 0, arr.length - 1);
        return 0;
    }
private:
    void mergeSort(int arr[], int left, int right) {
        
        if (left < right) {
            int mid = (left + right) / 2;
            mergeSort(arr, left, mid);
            mergeSort(arr, mid + 1, right);
            merge(arr, left, mid, right);
        }
    }
    
    void merge(int arr[], int left, int mid, int right) {
        
        int len = right - left + 1;
        
        int *tmp = new int[len]; // 可以先声明好
        
        int i = left; // 左边的头
        int j = mid + 1; // 右边的头
        int k = 0; // tmp 的头
        
        // 按照大小，将左右两边的装载到 tmp 中
        while (i <= mid && j <= right) {
            tmp[k++] = arr[i] <= arr[j] ? arr[i++] : arr[j++];
        }
        
        // 当一方到头了，将剩余的直接装载到 tmp
        while (i <= mid) {
            tmp[k++] = arr[i++];
        }
        
        while (j <= right) {
            tmp[k++] = arr[j++];
        }
        
        // 然后将 tmp 的所有内容复制到 arr
        k = 0;
        while (left <= right) {
            arr[left++] = tmp[k++];
        }
        
    }
};

优化想法

在 merge 的时候，tmp 临时数组需要不断分配内存，会影响效率，可以放在外部去分配，然后传入 merge 去使用即可。
