算法 - 快速排序

 Posted on July 21, 2018  |  2 minutes  |  296 words  |  Kim

快速排序的时间复杂度是 O(nlogn)，最坏情况是 O(n²)，最坏的情况就出现在给定数组是倒序排列

快速排序是不稳定排序，不稳定发生在 pivot 和 a[j] 交换的时刻

PS: 稳定排序的定义是若 arr[i] == arr[j]，排序之后第 i & 第 j 个元素的 前后顺序不变。

解题思路

通过递归的方法，逐步找到元素 i 的正确位置，找到该正确之后再将其移动到正确位置，再对 i 分成左右两个数组，然后递归

那么如何找到 i 元素的正确位置？

    1. 先定义左指针 i，右指针 j，分别指向这次递归数组的第 0 个元素和最后一个元素。

    i = left

    j = right

    1. 定义对比的基准点 pivot，一般取 pivot = arr[left]， 即这次递归的首个元素。需要注意的是，这一次递归就是要找到这个 pivot 的 正确位置，并将其移动到该位置

    pivot = arr[left]

    1. 开始递归：

    • 递归参数：

      • a、 arr 要排序的数组（原数组

      • b、 left 这次递归的起点

      • c、 right 这次递归的终点

    • 递归出口： left > right，即 left & right 相遇

      while (left > right)

    • 递归操作：

      • a、 循环，直到 i == j，即 i & j 相遇

        while (i != j)

      • b、 将 右指针 j 不断向左移动， 直到 指向的数小于等于 pivot，然后把最后指向的值 丢给 i 的位置（丢到左边）

            while (arr[j] > pivot && i < j) {
        j--;
    }
    arr[i] = arr[j];

      • c、 将 左指针 i 不断向右移动， 直到 指向的数大于等于 pivot，然后把最后指向的值 丢给 j 的位置（丢到右边）

            while (arr[i] < pivot && i < j) {
        i++;
    }
    arr[j] = arr[i];

      然后一直循环。。。。。 直到 i == j，即 i & j 相遇

      此时的位置 i （也就是 j）就是当前 pivot 的正确位置了

      • d、 然后将 pivot 赋值到 i 位置

        arr[i] = pivot

      • e、 最后进行数组的分割 & 递归调用

        数组的分割要以 i 为中点，最后分成两段：

        [left, i - 1] & [i + 1, right]

        quickSort(arr, left, i - 1);
quickSort(arr, i + 1, right);

完整代码 （C++）

class Solution {
public:
    int sortArray(int arr[]) {
        quick(arr, 0, arr.length - 1);
        return 0;
    }
private:
    void quick(int arr[], int left, int right) {
        
        if (left > right) return;
        
        int i = left, j = right;
        int pivot = arr[i];
        
        while (i != j) {
            
            // move until arr[j] > base
            while (arr[j] >= pivot && i < j) {
                j--;
            }
            arr[i] = arr[j]; // 丢到左边
            
            // move until arr[i] < base
            while (arr[i] <= pivot && i < j) {
                i++;
            }
            arr[j] = arr[i]; // 丢到右边
        }
        
        // 丢 pivot 到右边
        arr[j] = pivot;
        
        quick(arr, left, i-1);
        quick(arr, j+1, right);
        
    }
};
