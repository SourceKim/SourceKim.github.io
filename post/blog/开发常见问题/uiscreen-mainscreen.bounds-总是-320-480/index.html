[UIScreen mainScreen].bounds 总是 320 480

 发表于 May 6, 2020  |  1 分钟  |  62 个字  |  Kim

今天遇到一个问题：打印 [UIScreen mainScreen].bound 的时候总是得到 320 X 480。

po [UIScreen mainScreen].bounds
(origin = (x = 0, y = 0), size = (width = 320, height = 480))

原因是删除 SceneDelegate 和 Main StroryBoard 相关的问题时候，将 工程设置中的 Launch Screen File 置位了空

将其设置为 非空即可：比如 l，或者保留原先的 LaunchScreen 都行

(lldb) po [UIScreen mainScreen].bounds
(origin = (x = 0, y = 0), size = (width = 414, height = 896))
