HTTP2.0

相比 HTTP/1.X 的优势

  1. 采用二进制格式传输 Frame，而 1.0 使用的是文本传输

  2. 对 header 进行压缩

  3. 多路复用，可以实现真正的并发。之前只能通过 pipeline，但是多个请求之前的响应还是会被堵塞

  4. 服务器推送。服务器可以主动推送 JS、css 等资源，无需等待客户端解析的时候再去请求。
