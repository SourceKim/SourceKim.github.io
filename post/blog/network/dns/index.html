域名系统 （Domain Name System，DNS）

域名系统 （Domain Name System，DNS），主要用来：将域名（www.xx.com）转换到 ip 地址，是一个分层、分布式的数据库。

  • 运行在 UDP 上，使用 53 端口

  • 通常被其他 应用层协议 所用，如 HTTP SMTP

为什么要分布式、分层？

  1. 如果只有一台，容易发生单点故障

  2. 能减轻每台服务器处理的压力

  3. 能够让全世界用户都有更加近的服务器可以用，距离太远延迟很高
